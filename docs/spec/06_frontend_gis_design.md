# 06. 前端 GIS 设计

版本：v0.2  
日期：2026-05-06

## 1. 技术栈建议

```text
地图：MapLibre GL JS
曲线：ECharts 或 Plotly
状态管理：Pinia/Zustand/Redux 任选
前端框架：Vue 3 或 React
瓦片：Vector tiles + Raster/COG tiles
API：REST JSON
```

## 3. 底图

必须支持三种切换：地形、影像、矢量。

## 4. 图层分类

### 4.1 边界图层

- 一级流域边界。
- 二级流域边界。
- 当前启用 basin_version。
- 历史 basin_version，可选。

### 4.2 气象图层

- 降雨格点。
- 温度格点。
- 气象代站。
- best_available 气象产品。

### 4.3 水文图层

- 河段径流 `q_down`。
- 河段水位 `stage`。
- 洪水重现期。
- 预警等级。

## 5. 时间轴逻辑

前端不能生成固定小时序列。每个图层加载后调用：

```http
GET /api/v1/layers/{layer_id}/valid-times
```

并使用返回的 `valid_times[]` 控制时间轴。切换图层时，时间轴自动切换到新图层的有效时间列表。

## 6. Scenario 控件

至少支持：GFS、IFS、GFS + IFS 对比、Best Available。

## 7. 河段交互

### 7.1 视图切换逻辑

从全国总览进入流域详情的切换路径：

- 方式一：在全国总览页面点击流域信息弹窗的「进入分析」按钮
- 方式二：在水文预报 Tab 下，左侧流域列表选择目标流域
- 切换后地图自动 flyTo 至该流域 bbox，zoom level 提升至 7–9
- 左侧面板自动切换为流域级河段列表

### 7.2 流域详情视图（效果图 2）

进入流域视图后，河网根据当前选中水文变量（径流量或重现期等级）进行色标渲染。

**左侧面板 — 流域与河段：**
- 顶部：当前流域名称、basin_version_id
- 流域内河段列表：支持搜索和按预警等级筛选
- 河段列表每行显示：河段名称、当前 Q、重现期等级色标

**中央地图：**
- 河网按径流量/重现期等级色标渲染
- 流域边界高亮
- 关键城市和水文站标注

### 7.3 河段 hover 交互

hover 河段时：
- 高亮当前河段（线宽加粗 + 发光效果）
- tooltip 显示简要信息：

```text
河段名称：长江干流-宜昌至城陵矶
河段 ID：riv_000123
当前流量：5,242 m³/s
重现期：10–20 年一遇（警戒）
```

### 7.4 河段 click 交互

click 河段时，左侧弹出河段详情面板，内容包括：

```text

  当前径流:            5,242 m³/s
  水位变化:            +42.7 cm（较 24h 前）
  重现期等级:          10–20 年一遇（警戒）

  资料源:              GFS
```

面板底部提供操作按钮：
- 「查看详情」— 进入河段预报曲线详情全屏页面（§7.6）
- 「对比预报」— 在当前面板内叠加 IFS 曲线

**右侧面板 — 径流趋势小图：**

选中河段后，右侧面板显示该河段近 48 小时径流趋势迷你折线图（sparkline），标注当前时刻值和变化方向箭头，点击可跳转至全屏详情页。

### 7.5 右侧运行状态面板

右侧面板下方持续显示：
- 径流量/重现期图例（同 §9）
- 当前预报运行状态摘要（预报周期数、超警河段数）
- 最新更新时间

#### 7.6.1 顶部统计摘要条

横向排列四个关键指标卡片：

| 指标 | 示例值 | 说明 |
|---|---|---|
| 当前径流 | 3,860 m³/s | 最新时刻模拟/预报流量 |
| 水位变化 | +42.7 cm | 较 24 小时前的变化量 |
| 预报峰值 | 5,246 m³/s | 未来 7 天预报最大值及对应时刻 |
| 重现期 | 10–20 年一遇 | 峰值对应的重现期等级 |

在统计摘要条的右侧嵌入河段位置缩略图（120×90px），展示当前河段在流域中的地理位置：
- 流域边界灰色轮廓
- 当前河段红色高亮标注
- 不可交互，仅作定位参考

#### 7.6.2 左栏 — 气象代站列表与 forcing 图表

- 顶部：该河段关联的气象代站列表（station_id、名称、经纬度）
- 选中站点后展示该站 forcing 时序小图：
  - 降水（PRCP）柱状图
  - 温度（TEMP）折线图
- 支持切换不同代站查看 forcing 差异
- 标注数据来源（GFS / ERA5 / Best Available）

#### 7.6.3 中央主图 — 多源预报对比曲线

**核心图表规格：**
- 横轴：时间（过去 7 天 + 未来 7 天）
- 纵轴：流量 Q（m³/s）
- Analysis 段（实线）：scenario = analysis_true_field，蓝色
- GFS Forecast 段（实线）：scenario = forecast_gfs_deterministic，橙色
- IFS Forecast 段（虚线）：scenario = forecast_ifs_deterministic，绿色
- Analysis/Forecast 分界线：垂直虚线 + 标注「当前时刻」

**频率阈值线叠加：**
- Q2/Q5/Q10/Q20/Q50/Q100 水平线，从低到高颜色由浅蓝渐变至深红
- 每条阈值线右端标注 "Q5"、"Q10" 等标签
- 当前预报峰值所在区间高亮

**交互：**
- 图表顶部提供 Scenario 复选框组：☑ Analysis ☑ GFS ☐ IFS，勾选/取消控制曲线显示
- hover 显示十字准线 + tooltip（时间、各 scenario 对应 Q 值）
- 支持框选放大和双击重置

#### 7.6.4 右栏 — 洪水频率曲线与气象要素详情

**洪水频率曲线图：**
- 横轴：重现期 T（年），对数刻度
- 纵轴：流量 Q（m³/s）
- 绘制拟合曲线 + 置信区间带
- 标注当前预报峰值在曲线上的位置

**气象要素详情图组（纵向堆叠 3 个小图）：**
- 图 1：降水（PRCP）— 柱状图，标注累计降水量
- 图 2：温度（TEMP）+ 相对湿度（RH）— 双轴折线图
- 图 3：风速（wind）+ 气压（Press）— 双轴折线图
- 时间范围与主图同步，hover 联动

#### 8.4.2 多变量时序图表

纵向堆叠三组图表，时间范围与底部时间轴同步：

**图表 1 — 降水（PRCP）：**
- 柱状图，蓝色
- Y 轴：mm/day
- 标注统计值：24h 累计、7d 累计

**图表 2 — 温度（TEMP）+ 相对湿度（RH）：**
- 双 Y 轴折线图

每个变量旁显示质量标识：
- ✓ 正常：数据完整
- ⚠ 部分缺失：标注缺失时段

底部汇总：数据完整率百分比、最近一次 QC 时间。

### 8.5 底部时间轴

- 与全局时间轴逻辑一致（§5）
- 切换时刻时，右侧图表高亮对应时间点
- 时间分辨率按当前 forcing 的 native_time_resolution 显示

## 8B. 气象代站空间展示

气象展示收敛到单图 `/` 的「气象代站」图层。系统不再提供气象栅格图层、PNG 气象瓦片或独立气象空间展示页。

### 8B.1 图层入口

- `/meteorology` 作为旧入口重定向到 `/?metStations=1`；历史
  `?layer=met-stations` 链接按兼容规则归一化为流量图层 + 气象代站叠加。
- 图层切换器保留水文主图层选择，并以独立开关展示「气象代站」叠加；不展示气象栅格占位项。
- 低缩放级别展示代站聚合点；点击聚合点应放大到下一层聚合或具体站点。

### 8B.2 站点详情

- 点击站点打开与流量曲线窗口风格一致的玻璃弹窗。
- 同一个要素图幅同时展示 GFS 与 IFS 序列。
- PRCP、TEMP、RH、wind、Rn 通过同一弹窗内要素切换查看，只保留一个曲线图幅。

## 9. 图例设计

### 9.1 径流图层

可按绝对流量分级或按流域内分位数辅助配色。全国默认不建议仅按绝对 Q 配色，因为大江大河会压制中小流域变化。

### 9.2 重现期图层

```text
< 2 年       常态
2–5 年       偏高
5–10 年      关注
10–20 年     警戒
20–50 年     高风险
50–100 年    严重
> 100 年     极端
```

## 11. 前端状态模型

```typescript
// 全局状态
interface AppState {
  currentTab: 'overview' | 'meteorology' | 'hydrology' | 'flood_alert' | 'monitoring' | 'management';
  metSubTab: 'spatial' | 'station';
  userRole: 'viewer' | 'analyst' | 'operator' | 'model_admin' | 'sys_admin';
}

// 地图状态
interface MapState {
  activeBaseMap: 'terrain' | 'imagery' | 'vector';
  activeLayerIds: string[];
  activeScenarioIds: string[];
  activeValidTime: string;
  validTimes: string[];
  mapBounds: [number, number, number, number];
  zoomLevel: number;
}

// 流域与河段状态
interface HydroState {
  selectedBasinId?: string;
  selectedBasinVersionId?: string;
  selectedSegmentId?: string;
  segmentDetailExpanded: boolean;
  forecastDetailFullscreen: boolean;
}

// 气象状态
interface MetState {
  selectedStationId?: string;
  selectedMetVariable: 'PRCP' | 'TEMP' | 'RH' | 'wind' | 'Rn' | 'Press';
  selectedDataSource: string;
  rasterOpacity: number;
  showContours: boolean;
  showStationOverlay: boolean;
}

// 预警状态
interface FloodAlertState {
  alertThreshold: string;
  selectedAlertLevel?: string;
  alertSortBy: 'return_period' | 'flow' | 'basin';
}

// 监控状态
interface MonitoringState {
  selectedCycleTime?: string;
  selectedSource?: string;
  jobFilter: {
    status?: string;
    runType?: string;
    scenario?: string;
    modelId?: string;
  };
  jobSortBy: string;
  jobSortOrder: 'asc' | 'desc';
}

// 资产管理状态
interface ManagementState {
  selectedModelId?: string;
  selectedVersionId?: string;
  versionCompareMode: boolean;
  compareTargetVersionId?: string;
}
```

## 13. 洪水预警总览页面

### 13.1 页面描述

独立的预警总览视图，以全国地图为主体，河段按重现期等级分级着色（配色方案同 9.2 节）。该页面聚焦于当前及未来预报时段内超警河段的全局态势感知。

### 13.2 顶部预警信息条

滚动显示当前超警河段摘要，每条包含：

- 河段名称 / 所属流域。
- 预警等级（如 "警戒""高风险""严重""极端"）。
- 当前预报流量 Q（m³/s）。
- 对应重现期（如 T=25 年一遇）。

信息条自动轮播，点击任意条目可跳转至该河段详情。

### 13.3 左侧预警统计面板

按等级分组统计当前时刻超警河段数量：

```text
常态      (< 2 年)      ——条
偏高      (2–5 年)      ——条
关注      (5–10 年)     ——条
警戒      (10–20 年)    ——条
高风险    (20–50 年)    ——条
严重      (50–100 年)   ——条
极端      (> 100 年)    ——条
```

点击任意等级行，地图自动筛选并高亮该等级河段。

### 13.4 右侧 TOP 排名面板

按重现期严重程度降序排名的前 N 条河段列表（默认 N=20），列包括：

- 排名。
- 河段名称 / segment_id。
- 所属流域。
- 当前预报 Q。
- 重现期 T 值。
- 预警等级标签。

### 13.5 地图交互

- 河段按重现期等级着色，配色同 9.2 节。
- hover 高亮河段并 tooltip 显示河段名、Q、T 值。
- click 打开右侧详情面板，通过 generated client 请求 canonical route `/api/v1/basin-versions/{basin_version_id}/river-segments/{segment_id}/forecast-series`，携带 `basin_version_id` 与河段上下文，展示预报曲线及重现期阈值线。

### 13.6 时间控制

支持查看不同预报时刻的预警快照：

- 时间轴列出当前预报周期内所有有效时刻。

- 支持播放模式，自动逐时刻推进。

## 15. 产品监控与运行状态页面

### 15.1 页面描述

面向运维人员的流水线监控仪表盘，实时展示当前预报周期的作业执行状态、性能趋势和异常告警。

### 15.2 顶部摘要条

```text
当前起报周期：2026-05-04T00Z（示例）
总作业数：128
成功：110    失败：3    运行中：12    等待：3
Slurm 队列深度：15
```

各状态计数以对应颜色标识（成功绿、失败红、运行中蓝、等待灰）。

摘要条右侧嵌入 Slurm 队列深度环形图（donut chart），中央显示队列总深度数值，环形分段显示运行中（蓝色）、等待中（橙色）、空闲（灰色）的比例。

### 15.3 左侧流水线视图

当前周期各阶段状态卡片，按执行顺序纵向排列：

```text
download → canonical → forcing → shud_run → parse → frequency → publish
```

每个卡片显示：

- 阶段名称。
- 状态标识（成功 ✓ / 失败 ✗ / 进行中 ◉ / 等待 ○）。
- 耗时（已完成阶段显示实际耗时，进行中显示已用时间）。
- 流域完成率（如 85/128 = 66%）。

卡片之间以箭头连接，表示流水线依赖关系。

流水线视图下方显示各阶段耗时横向柱状图，每个阶段一条，横轴为耗时（分钟），颜色按阶段区分，柱状图末端标注具体耗时数值。

### 15.4 中间区域：作业列表表格

| 列名 | 说明 |
|---|---|
| run_id | 运行唯一标识 |
| model_id | 模型实例 ID |
| run_type | forecast / analysis / reforecast |
| scenario | GFS / IFS / best_available |
| status | queued / running / success / failed / cancelled |
| slurm_job_id | Slurm 作业 ID |
| submitted_at | 提交时间 |
| duration | 运行时长 |
| 操作 | 查看日志 / 重跑 |

支持按 status、run_type、scenario 筛选，按 submitted_at 或 duration 排序。

### 15.5 右侧区域：性能与成功率趋势

- **性能趋势图**：近 7 天各阶段平均耗时折线图，横轴为起报周期，纵轴为耗时（分钟），每条线代表一个阶段。
- **成功率趋势图**：近 7 天每个起报周期的作业成功率，横轴为起报周期，纵轴为成功率百分比。

### 15.6 状态机映射说明

forecast_cycle 状态到流水线阶段卡片的映射关系：

| forecast_cycle 状态 | 流水线阶段 |
|---|---|
| discovered / downloading | download |
| raw_complete | canonical |
| canonical_ready | forcing |
| forcing_ready | shud_run |
| forecast_running | shud_run |
| parsed_partial / parsed_complete | parse / frequency |
| published | publish |

前端根据 forecast_cycle 当前状态自动高亮对应的流水线阶段卡片。

### 15.7 操作权限

- operator 及以上角色可执行：触发单阶段重跑、取消运行中的作业。
- 普通用户仅可查看状态和日志，不可触发操作。
