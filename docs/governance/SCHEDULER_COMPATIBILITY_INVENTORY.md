## Authority

This page is a companion inventory for
`openspec/changes/governance-7-structural-entropy-controls/`. When it disagrees
with executable code, `services/orchestrator/scheduler.py` and the extracted
owner modules win. When it disagrees with governance policy,
`docs/governance/entropy-budget.md` and the OpenSpec change win.

The current compatibility facade exists because structural extraction has moved
state, lease, discovery, candidate-construction, evidence, and execution logic
out of `scheduler.py` while preserving old
`services.orchestrator.scheduler` import and monkeypatch paths.
