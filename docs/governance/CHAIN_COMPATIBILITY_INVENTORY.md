## Authority

This page is a companion inventory for
`openspec/changes/governance-7-structural-entropy-controls/`. When it disagrees
with executable code, `services/orchestrator/chain.py` and the extracted owner
modules win. When it disagrees with governance policy,
`docs/governance/entropy-budget.md` and the OpenSpec change win.

The current compatibility facade exists because structural extraction has moved
stage catalog, type, manifest, stage execution, array accounting, reservation,
retry, tile publication, worker-adapter, and persistence behavior out of or
around `chain.py` while preserving old `services.orchestrator.chain` import and
monkeypatch paths.
